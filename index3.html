Inicialización del repositorio localmente

Inicio de un nuevo repositorio en Git: con git init

Para crear un nuevo repositorio se usa el comando "Git init" es un comando que solo se utiliza una vez durante la configuración del repositorio nuevo. Al ejecutar este comando, se creara un nuevo subdirectorio ".git" en tu directorio en tu trabajo actual. También se creará una nueva rama principal.

Versión de el blog existente con un repertorio nuevo

En este caso, ya que tenemos el blog y ya contamos con la carpeta del proyecto, le damos click derecho dentro de la carpeta y le damos en abrir "open git bash here" y luego ejecutamos el comando "Git init".

Apuntar "Git init" a un directorio de proyecto existente hará que se ejecute la misma configuración de inicio mencionada arriba, pero en el ámbito de ese directorio.
